# Lender Fit Routing Copilot

Transparent funding-product routing for redacted borrower profiles.

- `api/` executable endpoints
- `actions/` GPT Action schemas and documentation
- `knowledge/current/` canonical 20-file Knowledge pack
- `lib/` shared routing and validation logic
- `site/` static HTML/CSS/JavaScript site
- `tests/` schema, privacy, action, and regression tests

This project does not guarantee approval or authorize automatic application submission.
